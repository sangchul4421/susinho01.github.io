# 프로젝트: 영수증 자동 파일명 변환기

## 목표
영수증 사진을 업로드하면 AI가 내용을 읽어서
20260614_스타벅스_카페_4500원.jpg 형식으로
파일명을 자동 생성하는 웹 앱

## 사용자
- 출장이 잦은 직장인 (주 사용자: 이상철)
- 매월 영수증 20~30장 증빙 필요
- 현재 수동으로 파일명 변경 중 → 월 3시간 낭비

## 파일명 형식
YYYYMMDD_가게명_업종_금액원.확장자
예시: 20260614_스타벅스_카페_4500원.jpg

## 현재 완성된 것
- index.html: 드래그 업로드 UI, 미리보기, 복사 버튼
- 파일명 생성은 날짜+시간만 (AI 연동 전)

## 다음에 만들 것

2. 가게명·업종·금액 자동 추출
3. 추출한 정보로 파일명 자동 생성
4. 실제 파일 다운로드 기능

## 기술 스택
- 프론트엔드: HTML, CSS, JavaScript (순수)
- AI: Claude Vision API (claude-sonnet-4-6)
- 배포: GitHub Pages
